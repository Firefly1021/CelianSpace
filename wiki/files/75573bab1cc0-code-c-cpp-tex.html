函数模版

1. 直接引入本地源文件

这里直接引入了当前目录下的 \texttt{main.cpp}。

\includeclist{./codeexamples/main.cpp}{快速排序主程序}

2. 只引入特定的函数片段

\includeclistpart{firstline=2, lastline=6}{./codeexamples/main.cpp}{核心算法逻辑}

通过参数，只展示文件中的核心逻辑（例如第 5 到 15 行）。

3. 手动插入简短代码

如果你不想创建文件，也可以直接在 LaTeX 中写：

#include <memory> #include <iostream>

void demo() { auto ptr = std::make_shared<int>(100); std::cout << "Value: " << *ptr << std::endl; }

在行内讨论变量 \cppin{std::vector<int> v;} 时也非常方便。
