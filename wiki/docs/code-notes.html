C++ 三目运算符

?: 可以作为简单 if-else 的表达式写法。

expression1 ? expression2 : expression3
int c = a > b ? a : b;

this 指针与 static

this 指向具体对象；静态函数属于类本身，不依赖某个实例。

void accelerate() { this->speed += 10; }
static bool isAdultRule(int check_age) { return check_age >= 18; }
